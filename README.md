# 📊 Kelompok-3_Business-Intelligence_UAS

## Multi-Platform Social Sentiment Evolution Analysis Dashboard


Dashboard **Business Intelligence** yang dikembangkan menggunakan **Microsoft Power BI** untuk menganalisis perkembangan sentimen media sosial, performa engagement, tren viralitas, persebaran wilayah, serta analisis konten dari berbagai platform media sosial.

Project ini bertujuan untuk mengolah data aktivitas media sosial menjadi informasi dan insight yang dapat mendukung proses **pengambilan keputusan berbasis data (Data-Driven Decision Making)**.

---

# 📌 Project Overview

Dashboard ini dirancang untuk memahami pola aktivitas pengguna, persepsi masyarakat, serta performa konten pada berbagai platform media sosial.

Analisis yang dilakukan meliputi:

- Overall Social Media Performance
- Sentiment Analysis
- Engagement Analysis
- Viral Trend Analysis
- Platform Comparison
- Geographic Analysis
- Content Intelligence
- Forecasting Analysis

Melalui dashboard ini, pengguna dapat memperoleh insight mengenai perkembangan sentimen masyarakat, performa platform media sosial, distribusi aktivitas berdasarkan wilayah, hingga prediksi tren engagement dan sentimen di masa mendatang.

---

# 📂 Dataset Information

| Information | Description |
|-------------|-------------|
| **Dataset** | Multi-Platform Social Sentiment Evolution Dataset |
| **Total Records** | 150,000+ Records |
| **Platform** | Facebook, Instagram, Twitter (X), TikTok, Reddit, YouTube |
| **Period** | 6 Months |
| **File Format** | CSV (.csv) |
| **Data Type** | Social Media Analytics |

Dataset berisi berbagai atribut penting yang digunakan dalam proses analisis, antara lain:

- **Platform** – Platform media sosial tempat konten dipublikasikan.
- **Topic** – Topik atau kategori pembahasan konten.
- **Location** – Lokasi atau wilayah pengguna.
- **Media Type** – Jenis media yang digunakan.
- **Engagement** – Total interaksi pengguna terhadap konten.
- **Views** – Total tayangan konten.
- **Viral Coefficient** – Indikator tingkat penyebaran konten.
- **Positive Sentiment** – Nilai sentimen positif.
- **Neutral Sentiment** – Nilai sentimen netral.
- **Negative Sentiment** – Nilai sentimen negatif.

Dataset dimanfaatkan untuk melakukan:

- Sentiment Analysis
- Engagement Analysis
- Viral Trend Analysis
- Geographic Analysis
- Platform Performance Analysis
- Forecasting Analysis

# ▶️ How to Use

1. Clone atau download repository ini.
2. Buka file **Kelompok 3_Business Intelligence_UAS.pbix** menggunakan **Microsoft Power BI Desktop**.
3. Pastikan file **multi_platform_social_sentiment_evolution.csv** berada pada folder **Dataset**.
4. Jika diperlukan, sesuaikan kembali lokasi data source.
5. Klik **Refresh** untuk memuat data.
6. Jelajahi dashboard menggunakan slicer dan filter yang tersedia.
